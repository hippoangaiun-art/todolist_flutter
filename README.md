# BUPT ToDo List

A powerful toy of TODO-List

仅作为移动应用实践训练课程作业  
基于Flutter开发

## 当前功能
1. 待办支持截止日期时间录入（日期 + 时分），并按“全部/按日期”双模式查看。
2. 待办主页支持标题搜索、完成状态筛选、已完成事项折叠分组。
3. 课表支持周网格查看、节次配置、课程多时段编辑、第一周日期设置。
4. 课表导入支持 `.xls` 与 `.xlsx` 两种格式。
5. 主题支持浅色 / 深色 / 跟随系统。
